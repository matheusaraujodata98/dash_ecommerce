# Projeto: Dashboard de Estatísticas de E-commerce

## Descrição
Aplicação web interativa desenvolvida com **Dash** para visualização dos dados do arquivo `ecommerce_estatistica.csv`.

## Execução local
1. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
2. Execute o app:
   ```bash
   python dash_ecommerce_app.py
   ```
3. Acesse no navegador:
   [http://127.0.0.1:8050](http://127.0.0.1:8050)

## Estrutura do projeto
```
├── dash_ecommerce_app.py
├── ecommerce_estatistica.csv
├── requirements.txt
└── README.md
```
